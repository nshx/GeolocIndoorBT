## Running the application
>>python api.py

## URLS
HomePage: http://127.0.0.1:5000/

Get whole registered beacons: http://127.0.0.1:5000/api/v1/beacons/all

Get specific beacon: http://127.0.0.1:5000/api/v1/beacons?id=0
